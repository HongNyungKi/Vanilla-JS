img tag에서 속성 srcest은 브라우저에 제시할(사용할) 이미지들과 그 이미지들의 원본 크기를 지정한다. 사용방법은, 사용할 이미지를 사이즈 별로 2장 이상 준비하여 , srcset 속성에 작성한다. 이때, 이미지의 크기로 px가 아닌 w 혹은 x 단위를 입력해야 하며, 작은 크기 이미지부터 순서대로 입력해야 한다. * 브라우저에 의해 어떤 이미지가 사용되건간에 고정된 이미지 크기를 유지하려면 속성width를 추가한다. - 단위 w (width descriptor) 이미지의 원본크기(가로 너비)를 의미한다. ex) 400*300(px) 크기의 이미지의 w 값은 400w이다. - 단위 x (device pixel descriptor) 이미지의 비율 의도(배수)를 의미한다. w 단위를 x 단위로 바꿔 사용 할 수 있다. 대부분 w 단위를 사용한다. 예제) image { width: 500px; height: 500px; background-image: url("images/heropy_small.png"); background-repeat: no-repeat; } @media (min-width: 401px) { image { background-image: url("images/heropy_medium.png"); } } @media (min-width: 701px) { image { background-image: url("images/heropy_large.png"); } }
