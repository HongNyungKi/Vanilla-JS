일반적으로 반응형 웹에서 이미지를 지원하기 위해, '미디어쿼리'라고 부르는 css media rule( @media )에서 background-image 속성을 많이 사용하는데, 반응형 이미지를 처리하기 위해 뷰포트(viewport)의 크기부터 사용자 화면의 해상도 들 많은 환경을 교려해야 한다. 하지만 html img 의 srcest 과 sizes를 통해 쉽게 이미지의 크기를 설정하는 것만으로 대부분의 고려사항을 사용자 브라우저에 떠널 길 수 있다. - 속성 srcset: (최적화 된 조건의 이미지중에 출력될 이미지)브라우저에게 제시할 이미지 URL과 원본 크기의 목록을 정의한다. 즉, 브라우저가 사용자 환경에 맞는 가장 최적화의 이미지를 사용한다. 이때, src속성은 생략이 가능하다. - 속성 sizes: (최적화 조건)미디어 조건과 해당 조건일 때 이미지 최적화 크기의 목록을 정의
